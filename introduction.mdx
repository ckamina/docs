## Overview

El siguiente diagrama visualiza la integración de kamina Academy dentro de una aplicación web ó móvil.

<img
  className="hidden dark:block"
  src="/images/diagram.svg"
  alt="Hero Dark"
/>

1. Para cada usuario llamar al endpoint embed via (`POST`) ([Documentación](/api-reference))
2. El servicio retonra una URL única por usuario (ej.: (`https://[cliente].kamina.academy/token=123456789abcdefghijk`)
3. Embeber kamina Academy en la webview o iframe con URL única por usuario
